Ecto Migration 用法总结

Chris Young's Blog
2019-06-27

常用命令

  1. mix ecto.gen.migration

  1. mix ecto.migrate
    执行 migration

  2. mix ecto.rollback

  1. mix ecto.dump
    导出表结构

数据类型

与 PostgreSQL 搭配的常用类型：

  • integer
  • string
  • map 对应 pg 的 jsonb
  • text
  • float
  • json

migration 写法

主要是在 3 个函数中工作: up, down, change
down 是 rollback 时候执行的，change 同时兼具 up 和 down 的作用。
